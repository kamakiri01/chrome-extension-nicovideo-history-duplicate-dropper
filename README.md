# Nicovideo history unifier

## Usage

- Go to `chrome://extensions`
- enable Developer mode
- Load unpacked, choose this app dir
- Go to `https://www.nicovideo.jp/my/history`

### uninstall

- Go to `chrome://extensions`
- Find `Nicovideo history unifier`
- click `Delete`

## How to build

```
$ npm install
$ npm run build
```
